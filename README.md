# Pre-Requisites
install docker-compose https://docs.docker.com/compose/install/

# Usage
Start a cluster:

- docker-compose up -d

Destroy a cluster:

- docker-compose stop

# Small Tutorial

Use https://www.kafkatool.com/ with docker.host.internal to connect and view the Messages
